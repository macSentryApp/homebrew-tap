# Homebrew Tap for macSentry

This is the official Homebrew tap for [macSentry](https://github.com/macSentryApp/macSentry), an automated security auditing tool for macOS.

## Installation

```bash
# Add the tap
brew tap macSentryApp/tap

# Install macSentry
brew install macsentry
```

Or install directly without adding the tap:

```bash
brew install macSentryApp/tap/macsentry
```

## Usage

```bash
# Run a security audit
macsentry

# Launch the GUI
macsentry-gui

# Schedule daily audits
macsentry-install

# More options
macsentry --help
```
